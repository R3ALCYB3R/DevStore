## 🔒 DevHelper Access Protocol
To ensure security and stability, **DevHelper** uses a multi-stage initialization:

1. **The 10s Hold:** Tap and hold the "START" button for exactly 10 seconds to unlock the interface. Letting go early will reset the connection.
2. **Access Methods:**
   * **💻 Computer:** Connect to the Windows/Mac Activator via USB.
   * **📄 Pairing File:** Upload a `.mobiledevicepairing` file for untethered refreshes.
   * **🚀 DevHelper Hub:** Enter the main dashboard to manage JIT and app updates.

> **Note:** If the app is force-closed, you must re-verify via the Computer or Pairing File to maintain your sideloaded status.
